asp.net中级软件开发工程师

工作经验:2年~3年 岗位职责：
1. 参与Hr系统关业务的开发. 2. 对完成的编码模块写单元测试. 3. 对复杂业务写详细设计文档.
要求:
1. 掌握ASP.NET、ASP.NET MVC,WebAPI,nhibernate. 2. 熟练掌握ms sqlserver. 3. 熟悉javascript,jquery,css. 4. 熟悉Scrum管理流程优先. 5. 熟悉saas优先. 6. 熟悉SOA架构优先.

UI设计师

岗位要求:
1、美术、设计或相关专业学历，1年以上设计相关工作经验，有独立项目设计、完稿经验； 2、熟练使用各种平面美术设计软件，具有良好色彩控制能力，精通界面切图处理技术； 3、熟悉Android、IOS等手机平台特性及相关平台下APP界面制作要求； 4、有一定文字功底，能通过文字补充和延伸美术产品的内涵 5、有基于用户体验的思考方式和良好的理解能力； 6、品行端正，充满热情，有敬业精神和团队精神； 7、具有较强的抗压能力及自我学习能力；
岗位职责：
1、负责参与移动产品的前期视觉用户研究、设计流行趋势分析，主导设定整体视觉风格； 2、准确理解产品需求和交互原型，设计出优质用户体验的界面效果图； 3、负责设计后期实现过程中的跟进与排查； 4、基于界面的静态概念设计，设计并制作动态演示效果； 5、负责分享设计经验、推动提高团队的设计能力。

Windows Phone开发工程师

职位描述:
1.负责xamarin 跨平台应用程序的开发； 2.评估开发计划执行情况，提交测试包给测试组进行测试 3.与测试组产品相关人员沟通，获取BUG文档，分析BUG并制定修改计划
任职要求：
1.具有一年以上WP平台应用开发经验； 2.有扎实的C#基础，了解应用基本算法/数据结构，了解基础设计模式； 3.熟悉windowsphone SDK 开发环境及相关开发工具； 4. 有xamarin 开发经验的优先考虑。

高级测试工程师（兼测试主管）

岗位职责:
1. 负责人力资源架构Cloud平台的产品测试 2. 建立规范化、制度化的测试流程及团队系统体系 3. 负责组内人员培养和培训工作 4. 测试过程管理与监控 5. 阶段性提交测试报告至PMO并进行工作汇报
任职资格:
1. 计算机相关专业,专科以上学历 2. 三年以上软件行业经验 3. 了解系统测试工作,掌握测试理论和测试方法 4. 熟悉和掌握功能、性能等测试工具和方法 5. 能够准确理解产品需求,编写测试计划，编写测试用例 6. 能独立测试结果分析和编写测试报告 7. 精通一种以上的自动化测试工具 8. 熟悉互联网产品使用及测试经验 9. 有良好的团队合作精神和沟通能力 10. 具有良好的人际沟通及语言表达能力，注重团队精神 11. 有SAAS,HR系统或其他管理系统行业经验优先

Web前端

工作职责：
1. 参与公司web前端的架构设计和标准和规范制定； 2. 利用各种Web技术实现web产品的交互效果，参与项目开发； 3. 对产品经理、设计师提出的需求给出技术评估和解决方案；
任职要求：
1. 精通HTML和CSS 及JavaScript、熟悉前端mvc/mvvm开发框架（例如backbone/angularjs等）优先； 2. 熟练使用前端技术开发单页面应用（SPA）； 3. 熟练解决页面兼容性问题； 4. 有jQuery框架使用经验，对ECMA规范有一定了解； 5. 熟悉服务端开发语言(例如C#/Java/PHP等)优先。

销售客户代表

岗位职责:
1.了解行业动态,分析市场需求; 2.熟悉公司产品及业务,落实公司政策,明确销售目标; 3.收集客户资源,联系客户,了解客户需求; 4.通过电话沟通,实地拜访,业务接待等方式,宣传推广公司产品,为客户策划详尽的解决方案; 5.保持与客户的沟通联系,处理客户异议,促进合作关系,推动销售目标的达成; 6.维护客户关系,跟进售后服务,提升客户满意度,促进二次合作;
岗位要求:
1.20-30周岁，形象好，气质佳，女性优先 2.大专及以上学历,专业不限; 3.热爱销售,激情澎湃,积极乐观,勤奋好学,充满自信,用于挑战自我; 4.有良好的业务交流沟通能力和具备一定的抗压能力; 5.有良好的职业素养和较强的团队精神观念; 6.有成功的销售经验者以及互联网工作经验者优先考虑;

市场推广及策划专员

岗位职责：
1、策划线下市场活动： 中大型巡展、小型会议的策划及安排、软、硬广告策划 2、线上市场活动： 付费、免费线上推广和宣传 3、市场调研与数据分析： 为管理者提供决策依据，为渠道选择提供数据支持 4、和相关行业策划市场合作及推广活动
任职要求：
1、24-30周岁，形象好、气质佳、女性优先 2、大专或以上学历，一年以上相同行业工作经验。 3、有较强的营销意识、沟通能力；充满激情，踏实、肯干。 4、积极主动，能承受压力，有独立工作能力和业务拓展能力。
